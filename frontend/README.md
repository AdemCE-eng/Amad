# Nadeem customer application

The Nadeem frontend is a React and Vite experience for personal saving, Saqr progression, family goals, rewards, notifications, and personalized saving opportunities.

## Run locally

From the repository root, use `run-project.bat` to start the complete demo. To run only the frontend:

```powershell
cd frontend
npm install
npm run dev
```

The default local address is `http://localhost:5173`.

## Verification

```powershell
npm test
npm run build
```

All demo financial and recommendation data is explicitly MOCK / SYNTHETIC.
